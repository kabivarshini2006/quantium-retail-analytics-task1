# Quantium Virtual Internship - Task 1  
**Retail Strategy and Analytics (Chips Category)**

## Project Overview
Completed **Task 1** of the Quantium Virtual Internship in Python.  
Analysed **264,000+** chip transactions to understand customer purchasing behaviour and delivered a data-driven strategic recommendation to the Category Manager (Julia).

## Key Objectives
- Perform data cleaning and exploratory data analysis (EDA)
- Identify high-value customer segments using LIFESTAGE and PREMIUM_CUSTOMER
- Calculate key metrics: total sales, units per customer, average price per unit
- Conduct statistical testing and affinity analysis
- Provide actionable commercial recommendations

## Skills Demonstrated
- Data Cleaning & Feature Engineering (pack size & brand extraction)
- Exploratory Data Analysis (EDA)
- Customer Segmentation
- Statistical Hypothesis Testing (independent t-test)
- Affinity Analysis (brand & pack size preference)
- Data Visualisation (Seaborn + Matplotlib)
- Strategic Business Recommendation

## Key Insights & Recommendation
- **Budget Older Families**, **Mainstream Young Singles/Couples**, and **Mainstream Retirees** drive the majority of chip sales.
- Mainstream young and mid-age singles/couples pay **significantly higher** price per unit (p-value < 2.2e-16).
- This segment shows strongest affinity for **Tyrrells**, **Twisties**, and **Doritos**, and prefers **270g packs**.
- **Strategic Recommendation**: Increase visibility of Tyrrells and 270g packs in impulse zones targeting young mainstream customers, while ensuring value multipacks for older families.
